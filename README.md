<h1>Meu Robô Assistente</h1>
<p>Aplicação desenvolvida para consultar os resultados das buscas do robô e permite registar novos filtros de busca.</p>

<h3>Funcionalidades</h3>
<ul>
  <li>Registo de Filtros: Os inputs são o título do anúncio e a url do OLX.</li>
  <li>O Robô busca (periodicamente) os resultados dos filtros, notifica os utilizadores por e-mail e também permite o utilizador consultar os anúncios encontrados. </li>
</ul>
